# Cursor AI 기반 커뮤니티 프로젝트

간단한 커뮤니티 기능을 구현한 웹 애플리케이션입니다.  
React와 Tailwind CSS로 UI를 구성하고, Axios를 통해 백엔드 API와 통신합니다.

이 프로젝트의 대부분의 코드 구현은 **Cursor AI**를 활용하여 생성되었습니다.
